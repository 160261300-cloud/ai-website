Resources

The tools, websites, and books I used while building this site and researching AI

Web Development

📚

W3Schools

My most-used reference while building this site. Whenever I wasn't sure about a CSS property or HTML tag, W3Schools had a clear explanation with a live example I could test instantly.

→ www.w3schools.com
🎨

Colour Hunt

I used Colour Hunt to find the dark blue colour scheme for this site. Having a consistent palette makes the site look professional rather than random.

→ colorhunt.co
🖼️

Freepik

A large library of free images and illustrations. Useful for sourcing graphics without copyright issues when building web projects.

→ www.freepik.com
📐

Draw.io

I used draw.io to sketch my wireframes before writing any code. Planning the layout first saved me a lot of time because I knew what I was building before I started.

→ app.diagrams.net

AI Learning

🤖

Elements of AI – University of Helsinki

A free course that introduced me to AI concepts without needing advanced maths. I recommend it to anyone starting out with AI — it explains things clearly and is genuinely interesting.

→ elementsofai.com
🧑‍💻

Google AI Education

Free resources from Google covering machine learning and responsible AI. Useful for going beyond the basics once you understand the fundamentals.

→ ai.google/education
📖

Books I Referenced

McGrath, M. (2020). HTML, CSS & JavaScript in Easy Steps — great for beginners.
Foster, J. (2022). HTML and CSS For Beginners — clear explanations with good examples.
Frain, B. (2020). Responsive Web Design with HTML5 and CSS — helped me understand how to make sites work on mobile.
